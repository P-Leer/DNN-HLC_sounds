Hearing Loss Compensation Audio Examples

Paper Systems

SNR9

Unprocessed

This is the original noisy audio at an SNR of 9 dB without any processing.

HLC_half

Processed using the Half-Gain Hearing Loss Compensation (HLC_half) system, applying half the prescribed gain based on the audiogram.

HLC_full

Processed using the Full-Gain Hearing Loss Compensation (HLC_full) system, applying the full prescribed gain.

NR_NAL_R

Processed first with a Noise Reduction algorithm and then with the NAL-R filter.

HLC-NR

Processed with the HLC_half system combined with Noise Reduction.

DNN-HA

Processed using the Deep Neural Network Hearing Aid (DNN-HA) system.

Ablation Group Systems

SNR9

HLC_ic_half

Ablation study variant using HLC with incomplete compensation (half-gain).

HLC_ic_full

Ablation study variant using HLC with incomplete compensation (full-gain).

NR_IC

Ablation study variant using Noise Reduction for the Incomplete Compensation model.
