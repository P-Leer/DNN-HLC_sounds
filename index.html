Hearing Loss Compensation Audio Examples

Paper Systems

SNR0

Unprocessed

This is the unprocessed audio at an SNR of 0 dB.

Audiogram Divided by 2

This system is derived from the audiogram divided by 2 in the log domain of the auditory model, applied at an SNR of 0 dB.

Full-Gain System

This system uses the full audiogram without division in the log domain, applied at an SNR of 0 dB.

NR_NAL_R

Processed with Noise Reduction and NAL-R filtering at an SNR of 0 dB.

HLC-NR

Processed using the audiogram divided by 2 in the log domain with Noise Reduction at an SNR of 0 dB.

DNN-HA

Processed using the Deep Neural Network Hearing Aid (DNN-HA) system at an SNR of 0 dB. More details available at https://github.com/fotisdr/DNN-HA.
